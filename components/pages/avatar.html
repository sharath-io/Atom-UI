Avatars

The Avatar component is used to represent a user, and displays the profile picture, initials or fallback icon.

Square Avatars

Add the avatar class to img element.Use the class avatar for avatar component and classes size-s size-m size-l for avatars of different sizes

Round Avatars

Included round class in the above example, here avatars appear in circle, that is achieved using border-radius: 50% Use the class avatar for avatar component and classes size-s size-m size-l for avatars of different sizes
