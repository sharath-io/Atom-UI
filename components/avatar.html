Avatars

The Avatar component is used to represent a user, and displays the profile picture, initials or fallback icon.

Square Avatars

Add the avatar class to img element. There are 3 additional sizes available, including avatar small, avatar medium, and avatar large.

Round Avatars

Included round class in the above example, here avatars appear in circle, that is achieved using border-radius: 50%
