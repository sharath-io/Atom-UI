Alerts

Alerts are used to provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.Alerts are available for any length of text, as well as an optional close button.

Status alerts

For proper styling, use one of the four required contextual classes : alert-primary, alert-success, alert-danger, alert-warning
